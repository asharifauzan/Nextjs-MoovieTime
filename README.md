## Getting Started

Install Dependencies
`npm install --force`

Run Application (ensure run in localhost with port 3000)
`npm run dev -- -H localhost --port 3000`

## Tech Stack

- Nextjs 15 + Typescript
- Shadcnui
- Tailwind
- Swiper (carousel)
